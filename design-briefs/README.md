# Design Briefs

This section contains design brief samples for social media visuals, infographics, campaign posts, and brand-aligned digital content.

## Purpose

The purpose of this section is to show my ability to plan visual content clearly before designing or assigning the design task.

A design brief helps explain the purpose of the visual, target audience, message, layout idea, colour direction, and content requirements.

## What This Section Will Include

- Social media design briefs
- Infographic planning notes
- Instagram post layout ideas
- Campaign visual direction
- Brand-aligned content ideas
- Canva design planning
- Photopea editing direction
- Visual content checklists

## Design Brief Focus

Each design brief will focus on:

- Design objective
- Target audience
- Main message
- Platform
- Suggested layout
- Visual direction
- Text to include
- Call-to-action
